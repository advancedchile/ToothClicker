# 🦷 Tooth Clicker

Un clicker/idle game dental desarrollado como experimento por **Atom Labs**.

👉 **Juega ahora:** [advancedchile.github.io/tooth-clicker]

## ¿Qué es?

Hacé click en el diente, comprá generadores, desbloqueá mejoras, subí al ranking global. Un clicker clásico con estética dental.

## Características

- 🎯 **120+ generadores** — de cepillos a dimensiones esmaltadas
- ⚡ **100+ mejoras de click** — bonos, multiplicadores, sinergias
- 🏆 **300+ logros** — automáticos y secretos
- 👑 **Sistema de prestigio** — reiniciá para un bonus permanente
- 🌐 **Ranking global** — compará tu progreso con otros jugadores
- 💾 **Progreso persistente** — se guarda local + cada jugador tiene su slot
- 🌍 **Español e inglés**
- 🔢 **4 formatos numéricos** — corto, largo, ingeniería, científico
- ⏱️ **Earnings offline** — gana dientes mientras no jugás (cap de 2h)

## Stack

HTML + React (via Babel standalone) + CSS con tokens de Atom Design System. Cero build, cero backend (ranking vía JSONBin). Funciona abriendo el HTML directamente.

## Desarrollo local

No hay build step. Clonás el repo y abrís `index.html` en el navegador (o serví con cualquier static server):

```bash
python3 -m http.server 8000
# abrí http://localhost:8000
```
